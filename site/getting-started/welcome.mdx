# Welcome to ChukMCPServer

ChukMCPServer is a **high-performance MCP (Model Context Protocol) server framework** for Python, designed for production-grade applications and seamless Claude Desktop integration.

## Why ChukMCPServer?

```python
from chuk_mcp_server import tool, run

@tool
def add_numbers(a: int, b: int) -> int:
    """Add two numbers together."""
    return a + b

run()  # Production-ready server in 5 lines!
```

## What is MCP?

The **Model Context Protocol (MCP)** is an open standard that enables AI assistants like Claude to securely connect to external data sources and tools. Think of it as USB for AI - a universal way to plug any tool into any AI assistant.

## Beyond Basic MCP

While MCP provides the protocol, ChukMCPServer adds:

- **Performance optimization** - <3ms overhead vs 5-50ms in typical implementations
- **Production features** - OAuth, rate limiting, health checks, metrics
- **Developer experience** - Zero config, auto-detection, FastAPI-like decorators
- **Deployment flexibility** - STDIO, HTTP, SSE, Docker, cloud platforms
- **Server composition** - Aggregate multiple servers with configuration
